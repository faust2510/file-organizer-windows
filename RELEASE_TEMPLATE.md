### 🚀 使用说明

**第一次使用？**

1. 下载上面的 `.zip` 文件
2. 解压到桌面（右键 → 全部解压缩）
3. 双击 `智能文件整理助手.exe`
4. 看到蓝色界面就成功了！

**操作步骤：**

1. 点「扫描文件」→ 查看要整理的文件
2. 确认没问题
3. 点「整理文件」→ 完成！

**如果出错了：**

- 点「撤销整理」可以恢复到原来的样子
- 文件不会被删除，只是移动到 `OrganizedFiles` 文件夹

## 使用方法

1. 复制上面「模板开始」到「模板结束」之间的内容
2. 替换以下占位符：
   - `vX.X.X` → 实际版本号（如 v1.0.0）
   - `2026年X月X日` → 实际发布日期
   - `[描述新功能]` → 实际更新内容
   - `下载链接` → 实际的 GitHub Releases 下载链接
3. 粘贴到 GitHub Releases 页面
4. 发布！

## 版本号规则

- **主版本号 (X.0.0)**：重大更新，界面大改或功能大变
- **次版本号 (0.X.0)**：新增功能，但整体不变
- **修订号 (0.0.X)**：修复小问题，优化性能

例如：v1.0.0 → v1.0.1（修复bug）→ v1.1.0（新功能）→ v2.0.0（大改版）
