电脑桌面太乱了？下载文件夹堆了几百个文件找不到？

**妈妈文件整理助手** 帮你一键整理！

- 📄 文档（Word、PDF、Excel……）
- 🖼️ 照片（JPG、PNG、HEIC……）
- 🎬 视频（MP4、MOV……）

## 怎么用？（3 步搞定）

### 第 1 步：下载

下载 `妈妈文件整理助手-Windows.zip`，解压到桌面。

### 第 2 步：打开

双击 `妈妈文件整理助手.exe`，看到蓝色的界面就对了。

### 第 3 步：整理

1. 点 **「扫描文件」** → 软件自动查找桌面上乱七八糟的文件
2. 看一眼列表，确认没问题
3. 点 **「整理文件」** → 完成！🎉

整理完后，你的文件会被分类到 `C:\Users\你的名字\OrganizedFiles` 文件夹里：

```
OrganizedFiles/
├── 文档/      ← Word、PDF、Excel 放这里
├── 图片/      ← 照片放这里
├── 视频/      ← 视频放这里
├── 音乐/      ← 音乐放这里
├── 压缩包/    ← ZIP、RAR 放这里
├── 安装包/    ← exe、msi 放这里
└── 其他/      ← 不认识的文件放这里
```

| 按钮 | 功能 |

| 🔍 **扫描文件** | 查找桌面上的文件，显示预览列表 |

## 常见问题

### Q: 我不会安装，怎么办？
A: 不需要安装！下载解压后直接双击就能用。

### Q: 整理后文件去哪了？
A: 在 `C:\Users\你的名字\OrganizedFiles` 文件夹里，按类型分好了。

### Q: 我想恢复原来的样子？
A: 打开软件，点「撤销整理」就行。

### Q: 软件会不会偷看我的文件？
A: 不会。软件只看文件名和后缀名（比如 `.jpg`、`.docx`），不会打开文件内容。而且完全离线，不联网。

### Q: 我的文件太多了，整理要多久？
A: 几百个文件通常 1 分钟内搞定。
